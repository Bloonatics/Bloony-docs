# Main Command: punch

## Description

Punches your opponent in the battle game.

## Syntax

```
[p]punch [member]
```

## Argumemts

###### member

- Type: [MemberResolvable](/typedefs/MemberResolvable.md)
- Optional: `true`
- Default: *`None`*

## Aliases

*`None`*
